RBAC for SkeekS CMS
===================================

Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist skeeks/rbac "*"
```

or add

```
"skeeks/cms-rbac": "*"
```
